# Setup React App on Mac

How to Setup React-JS App on Mac



>First, Install Node Package and NPM <br>


> After That Run Following Commands in Terminal<br>


  - npx create-react-app my-app
    - *my-app is a name of the App*
  - cd my-app
  - npm start
